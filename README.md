# Fitness-Calculator-System

نظام برمجى متكامل مبني باستخدام لغة C++ ومبادئ الـ OOP لإدارة وحساب النشاط الرياضي.

## ⚙️ المواصفات التقنية (Technical Specifications):
- **Object-Oriented Architecture:** تم بناء النظام باستخدام كلاسات منفصلة (Classes) لكل كيان (User, Activity, Analytics) لضمان سهولة التطوير.
- **Modular Design:** النظام مقسم إلى دوال (Functions) متخصصة، مما يجعل الكود نظيفاً (Clean Code) وقابلاً للصيانة.
- **Data Encapsulation:** حماية بيانات المستخدم من خلال استخدام الـ Private members وتوفير واجهة تحكم آمنة.

## 🚀 لماذا هذا النظام مميز؟
هذا المشروع ليس مجرد كود بسيط، بل هو "محرك" (Engine) يمكن ربطه بأي تطبيق موبايل أو موقع إلكتروني لحساب السعرات الحرارية. تم التركيز فيه على:
1. الدقة في المعادلات الحسابية.
2. التنظيم البرمجي العالي الذي يسهل التعديل عليه.
3. الأداء السريع بفضل الاعتماد على C++.

## 🛠 كيف تستخدمه؟
النظام مهيأ لإضافة أي نشاط رياضي جديد (مثل الجري، السباحة، أو رفع الأثقال) بمجرد إضافة كلاس جديد أو Function جديدة دون التأثير على عمل النظام الأساسي.
